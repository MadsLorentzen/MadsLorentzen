# Hi, I'm Mads 👋

I'm a **data scientist and ML engineer** based in the Greater Copenhagen Area. I hold a PhD in Geophysics from the University of Copenhagen / GEUS, where I built supervised learning pipelines for large seismic datasets. Today I work at the intersection of **machine learning, GenAI, and scientific computing**, turning messy real-world data into models, tools, and automated workflows.

## 🔭 What I'm working on

- **Machine learning & deep learning:** classical ML and neural networks (CNNs, feed-forward) for prediction and classification on tabular, image, and signal data
- **GenAI & LLM workflows:** prompt engineering, RAG, semantic search, NER, and fine-tuning with the OpenAI and Anthropic APIs
- **Agentic coding & automation:** hands-on with [Claude Code](https://www.claude.com/product/claude-code) to automate document processing, data extraction, and dev workflows
- **Applied data science:** inverse problems, Bayesian statistics, and end-to-end pipelines from raw data to deployed model
